# Terminal Icons

Place terminal application icons here. Supported formats: PNG, SVG, ICNS

Expected files:
- terminal.png (or .svg, .icns) - macOS Terminal.app
- ghostty.png (or .svg, .icns) - Ghostty
- alacritty.png (or .svg, .icns) - Alacritty
- kitty.png (or .svg, .icns) - Kitty
- iterm.png (or .svg, .icns) - iTerm2
- warp.png (or .svg, .icns) - Warp
