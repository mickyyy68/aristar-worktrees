# Terminal Icons

Place terminal application icons here. Supported formats: PNG, SVG

Expected files:
- terminal.png (or .svg) - macOS Terminal.app
- ghostty.png (or .svg) - Ghostty
- alacritty.png (or .svg) - Alacritty
- kitty.png (or .svg) - Kitty
- iterm.png (or .svg) - iTerm2
- warp.png (or .svg) - Warp
